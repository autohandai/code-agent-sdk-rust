# Autohand Code Agent SDK Code of Conduct

The Autohand Code Agent SDK community includes people building developer tools, integrations, automation, documentation, and examples across many languages and backgrounds.

We want these spaces to be useful, welcoming, and professional. This code applies to project repositories, issue trackers, pull requests, discussions, community channels, events, and other spaces managed by Autohand AI for the Agent SDK.

If you believe someone is violating this code of conduct, report it by emailing [hey@autohand.ai](mailto:hey@autohand.ai). For security-sensitive reports, email [security@autohand.ai](mailto:security@autohand.ai).

## Our Standards

- **Be friendly and patient.** Assume good intent and leave room for people to learn.
- **Be welcoming.** Support people of all backgrounds, identities, experience levels, and abilities.
- **Be considerate.** Your decisions, comments, and contributions affect other users and maintainers.
- **Be respectful.** Disagreement is expected; personal attacks and harassment are not.
- **Choose words carefully.** Do not insult, demean, threaten, harass, or exclude others.
- **Resolve disagreements constructively.** Focus on facts, tradeoffs, user impact, and paths forward.

Unacceptable behavior includes violent or discriminatory language, sexualized content, doxing, personal insults, repeated unwanted contact, or encouraging any of those behaviors.

## Enforcement

Maintainers may remove comments, issues, pull requests, or other contributions that violate this code. Maintainers may also temporarily or permanently limit participation for behavior that harms the project or community.

## Questions

For SDK documentation, visit https://autohand.ai/docs/agent-sdk/. For other questions, contact [hey@autohand.ai](mailto:hey@autohand.ai).

This code follows the same spirit as the Autohand Code CLI community guidelines and is adapted for the Agent SDK repositories.
